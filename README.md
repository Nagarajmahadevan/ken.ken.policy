# KenKen Puzzle Privacy Policy

This repository hosts a single-page privacy policy for Google Play using GitHub Pages.

## Steps to publish on GitHub Pages

1) Create a new GitHub repository (for example, `kenken-privacy-policy`).
2) Add the files:
   - `index.html`
   - `styles.css`
   - `README.md`
3) Replace placeholders at the top of `index.html`:
   - `<DEVELOPER_NAME>`
   - `<CONTACT_EMAIL>`
   - `<EFFECTIVE_DATE>`
   - `<NONE / LIST_SDKS>`
4) Commit and push to the `main` branch.
5) Enable GitHub Pages:
   - Repository Settings -> Pages
   - Source: "Deploy from a branch"
   - Branch: `main`
   - Folder: `/root`
6) Save. GitHub will build and publish the site.

## Live URL format

Your privacy policy will be available at:

https://<github-username>.github.io/<repo-name>/
